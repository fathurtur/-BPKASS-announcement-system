STASIUN MANDAI
Stasiun Mandai (MDI) adalah stasiun kereta api kelas kecil yang terletak di Jalan Poros Kima Maros, wilayah Dusun Cinranae, Desa Ma'rumpa, Kecamatan Marusu, Kabupaten Maros, Sulawesi Selatan, Indonesia. Stasiun Mandai merupakan stasiun yang letak nya dekat dengan Bandara Internasional Sultan Hasanuddin. Stasiun ini juga merupakan stasiun kereta api yang letak nya paling selatan di Kabupaten Maros. Stasiun Mandai terletak di jalur kereta api Trans-Sulawesi yang nantinya menghubungkan Makassar dan Parepare. Stasiun yang terletak pada ketinggian 11 mdpl ini menjadi stasiun terminus (ujung) selama jalur kereta api ke arah Makassar dalam tahap pembangunan. Stasiun Mandai yang terletak di antara Stasiun Parangloe (rencana) dan Stasiun Maros.
STASIUN MAROS
Stasiun Maros (MRS) (Lontara Bugis: ᨔᨛᨈᨔᨗᨕᨘ ᨆᨑᨘ , transliterasi: Sētasiun Maru' ; Lontara Makassar: ᨔᨙᨈᨔᨗᨕᨘ ᨆᨑᨘᨔᨘ , transliterasi: Sétasiun Marusu' ) adalah stasiun kereta api kelas besar yang terletak di wilayah Lingkungan Data, Kelurahan Pallantikang, Kecamatan Maros Baru, Kabupaten Maros, Sulawesi Selatan, Indonesia. Nama "Maros" yang digunakan sebagai nama stasiun ini diambil dari nama kabupaten tempat stasiun ini dibangun. Stasiun ini merupakan stasiun kereta api yang dekat dengan pusat ibu kota Kabupaten Maros. Stasiun Maros terletak di jalur kereta api Trans-Sulawesi yang nantinya menghubungkan Makassar dan Parepare. Stasiun Maros yanga berada di ketinggian 2 mdpl ini terletak di antara Stasiun Mandai dan Stasiun Rammang-Rammang.

STASIUN RAMMANG-RAMMANG
Stasiun Rammang-Rammang (RMG) (Lontara Bugis: ᨔᨛᨈᨔᨗᨕᨘ ᨑᨆ–ᨑᨆ , transliterasi: Sētasiun Rammang-Rammang ; Lontara Makassar: ᨔᨙᨈᨔᨗᨕᨘ ᨑᨆ–ᨑᨆ , transliterasi: Sétasiun Rammang-Rammang ) adalah stasiun kereta api kelas kecil yang terletak di Jalan Poros PT Semen Bosowa Maros, wilayah Dusun Rammang-Rammang, Desa Salenrang, Kecamatan Bontoa, Kabupaten Maros, Sulawesi Selatan, Indonesia. Nama "Rammang - Rammang" yang digunakan sebagai nama stasiun ini diambil dari nama dusun tempat stasiun ini dibangun. Stasiun ini merupakan stasiun kereta api yang lokasinya berada di paling utara Kabupaten Maros. Stasiun Rammang - Rammang terletak di jalur kereta api Trans-Sulawesi yang nantinya menghubungkan Makassar dan Parepare. Stasiun Maros yanga berada di ketinggian 1 mdpl ini terletak di antara Stasiun Maros dan Stasiun Pangkajene.

STASIUN PANGKAJENE
Stasiun Pangkajene (PKJ) adalah stasiun kereta api kelas besar yang terletak di wilayah, Male'leng, Desa Kabba, Kecamatan Minasatene, Kabupaten Pangkajene dan Kepulauan, Sulawesi Selatan, Indonesia. Nama "Pangkajene" yang digunakan sebagai nama stasiun ini diambil dari nama kabupaten tempat stasiun ini dibangun. Stasiun ini merupakan stasiun kereta api yang dekat dengan pusat ibu kota Kabupaten Pangkajene dan Kepulauan, dan menjadi stasiun yang letak nya paling selatan Kabupaten Pangkajene dan Kepulauan. Stasiun Pangkajene terletak di jalur kereta api Trans-Sulawesi yang nantinya menghubungkan Makassar dan Parepare. Stasiun yang berada diketinggian 1 mdpl ini, terletak di antara Stasiun Rammang-Rammang dan Stasiun Labakkang.
